NOCTIS v2 — by Shun

运行方式：
1. 双击 index.html 打开首页。
2. 点击“进入系统”进入真实多伦多市中心地图。
3. 地图页面需要联网，因为使用 OpenStreetMap 地图瓦片和 Leaflet。

如何添加你的摄影图片：
1. 进入 map.html。
2. 点击左下角 “+ 添加摄影点”。
3. 在地图上点击你的拍摄位置。
4. 弹窗出现后，点击“上传或替换照片”。
5. 写标题、摄影描述、灯光分析。
6. 点击“保存编辑”。
7. 点击“收藏”可加入收藏状态。

说明：
- 你新增的点和图片会保存在浏览器 localStorage 里。
- 如果换浏览器或清除缓存，新增内容会消失。
- 后续可以升级成真正数据库版本。
