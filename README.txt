NOCTIS v3 - noctis-lake

How to run:
1. Open index.html in a browser.
2. Click Enter Map to open the Toronto night light map.
3. The map page needs an internet connection because it uses OpenStreetMap tiles and Leaflet.

Features:
- Generative night-map landing page with project status and entry points.
- Map filters for All, Commercial, Residential, and Favorites.
- Search across titles, times, conditions, observations, and analysis notes.
- Add photo points, upload images, and edit area, condition, time, light intensity, observation, and analysis fields.
- Save favorite points, delete points, focus the current map results, and export a JSON backup.
- Map points sync to Supabase for cross-device viewing and public submissions.

Notes:
- Supabase project: noctis-lake.
- Browser localStorage is used only as an offline cache. The app keeps trying to sync when the network is available.
- The current version uses an open contribution model: any visitor can add, edit, or delete public map points.
